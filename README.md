## Run Locally

Clone the project

```bash
  git clone https://github.com/enjinerd/postfy.git
```

Go to the project directory

```bash
  cd postfy
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```

Access **http://localhost:3000**
